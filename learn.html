Where Does My Data Go?

Match your survey question type to the right analysis method

Radio Button / Single-response
One answer per question (e.g., "Which brand do you prefer?")
Data looks like: Brand A / Brand B / Brand C (one column, text or codes)
Use: Frequencies, Chi-Square
Use this method →
Rating / Likert Scale
Rate on a scale (e.g., "How satisfied are you? 1-5")
Data looks like: 4, 3, 5, 2, 4... (one column, integers)
Use: Summary Statistics, T-test, ANOVA, Correlation
Use this method →
Matrix / Rating Grid
Rate multiple items on the same scale (e.g., "Rate each feature 1-5")
Data looks like: Multiple columns, every cell has a rating
Use: Scale Comparison (Multi-Response), Cronbach's Alpha, Factor Analysis
Use this method →
Checkboxes / Multi-response
Select all that apply (e.g., "Which features do you use?")
Data looks like: Multiple columns, mostly blank, value = selected
Use: Multi-Response Analysis
Use this method →
NPS (0-10)
"How likely are you to recommend?" on a 0-10 scale
Data looks like: One column: 8, 6, 10, 9, 3...
Use: Summary Statistics (shows Promoter/Passive/Detractor breakdown)
Use this method →
Ranking
Rank items in order (e.g., "Rank these 5 features")
Data looks like: Multiple columns, each row is a permutation of 1-N
Use: Friedman Test, Repeated Measures
Use this method →
Binary (Yes/No)
Two outcomes (e.g., "Did the customer churn?")
Data looks like: One column: Yes/No, 1/0, True/False
Use: Frequencies, Chi-Square, Logistic Regression
Use this method →
Numeric / Continuous
Measurements with no fixed scale (e.g., revenue, age, time)
Data looks like: One column: 45.2, 1200, 67.8...
Use: Summary Statistics, Regression, Correlation
Use this method →
Open-ended Text
Free-text responses (e.g., "What could we improve?")
Data looks like: One column of text responses
Use: Word Frequency, Sentiment Analysis
Use this method →
Paired / Before-After
Same respondents measured twice (e.g., pre/post training)
Data looks like: Two columns: Before and After values for the same people
Use: Paired T-test, Wilcoxon
